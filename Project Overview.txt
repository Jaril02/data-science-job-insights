#Data Science Job Trends Analysis

This project analyzes real-world job posting data for Data Science roles using Python, NLP, and visualization tools.

# Features
- Job Title Grouping
- NLP-based Skill Extraction
- Top Hiring Companies
- Location-based Job Demand
- Interactive Dashboard (Streamlit)
